# React Native Social

Social feed app with posts, author profiles, likes, and comments. Built with expo-router, react-native-safe-area-context, and platform guards.

**Stack:** React Native | **Complexity:** moderate
**Tags:** `mobile`, `expo`, `react-native`, `social`, `feed`, `profiles`

## Quick Start

```bash
npm install
npx expo start
```
